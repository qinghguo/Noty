说明书：

1. 快捷键“Option + [“折叠成五角星；

2. 点击五角星展开便签；

3. 长按五角星关闭当前便签；

4. 快捷键”Option + /“可以把当前便签置顶于所有窗口之上，重复按键取消置顶；

5. Command + W 关闭展开状态的便签，Command + Q 退出软件。
